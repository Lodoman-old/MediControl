### Recomendaciones para capturas

- Resolución: 1280×720 píxeles mínimo
- Formato: PNG sin compresión
- Ocultar datos sensibles (si los hay)
- Capturar la pantalla completa del navegador
- Incluir bordes de ventana para contexto profesional
